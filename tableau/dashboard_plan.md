# Tableau Dashboard Plan

## Dashboard Title

Healthcare Revenue Integrity Analytics Platform

## Audience

Revenue integrity leaders, coding audit managers, compliance analysts, finance leaders, and healthcare data analytics recruiters.

## Executive KPIs

- Total Medicare services
- Total estimated Medicare payments
- Average submitted charge
- Average Medicare payment
- Charge-to-payment ratio
- Number of providers analyzed
- Number of specialties analyzed
- Number of HCPCS codes analyzed

## Planned Dashboard Views

1. Executive Overview
2. Specialty Comparison
3. HCPCS Payment Variance
4. Provider Outlier Review
5. Geographic Trends
6. Audit Prioritization Matrix

## Important Language

Use "risk indicator," "outlier," "variance," and "audit prioritization." Avoid saying the dashboard proves fraud, abuse, or incorrect billing.
